# Azin Mirror Website

Modern website for Azin Mirror, manufacturer of LED and decorative mirrors.

## About

Azin Mirror is a mirror manufacturing company specializing in:
- LED mirrors
- Bathroom mirrors
- Decorative mirrors
- Custom projects

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Motion

## Features

- Persian / English support
- Product showcase
- B2B project presentation
- WhatsApp customer support
- Responsive design

## Development

Install:

npm install

Run:

npm run dev

Build:

npm run build
